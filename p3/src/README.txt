Para utilizar este codigo es necesario que se ejecute con PYthon3 para evitar
errores de ASCII.

python3 TodosVsTodos.py 

Pedira que se ingresen alguns datos para generar un arreglo aleatorio
que se utilizara para los algoritmos:

QuickSort()
QuickRandomSor()
HeapSort()

Se imprimira en pantalla los resultados. ver "EJEMOLO.PNG"
